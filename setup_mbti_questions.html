🧠 Setup MBTI Questions

📋 Instructions

This tool will help you add MBTI questions to your database using the admin API.

  1. Make sure your backend is running on http://localhost:8000
  2. You need to be logged in as an admin user
  3. Get your admin auth token from the frontend
  4. Add sample MBTI questions using the button below

🔑 Admin Authentication

First, get your admin authentication token:

  1. Go to your frontend application (http://localhost:3000)
  2. Log in as an admin user
  3. Open Developer Tools (F12)
  4. Go to Application → Local Storage → http://localhost:3000
  5. Copy the value of "accessToken"
